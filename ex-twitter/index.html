What's Bad About Ex-Twitter

To access postings on ex-Twitter requires the user to run nonfree JavaScript code. Running that code damages your freedom and your security.

The API that used to permit reading postings on Twitter without logging in was shut off, so the proxy site nitter.net ceased to work.

Musk seems determined to adopt for Ex-Twitter every nasty thing that users have criticized web sites for.

The user is required to pay a trivial sum, but the surveillance implied by the payment is more important than the payment itself.
