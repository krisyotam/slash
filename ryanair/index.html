What's Bad About Ryanair

Ryanair has eliminated airport check-in, so it now requires every passenger to check in using an app running on a so-called "smart" phone. A snoop-phone running some non-Google Android variant won't work.

It does reportedly allow using someone else's snoop-phone to check in, but it is still added pressure on people to have one. Please, even if you do have a snoop-phone, boycott the airlines (and other companies) that exclude the people who don't have one.
